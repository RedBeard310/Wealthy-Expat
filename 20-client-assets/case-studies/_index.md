# Case studies

Specific past client wins with numbers, situations, interventions, outcomes.
The writing agent uses these to ground "we helped X get Y" claims with real
evidence.

## Conventions

- One case study per file. Filename: `YYYY-MM-anonymized-or-named-summary.md`
- Each file: situation (with specifics), what was done, the quantified outcome,
  any anonymization notes.
- If the client is named (with permission), say so. If anonymized, note what's
  been changed.

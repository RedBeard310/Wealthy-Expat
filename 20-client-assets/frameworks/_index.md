# Frameworks

Named methodologies the client owns. The writing agent uses these when scripts
need to lean on the client's signature systems (vs. inventing them).

## Conventions

- One framework per file. Filename: snake-case-name.md
- Each file: name, when to use, the steps/phases (with discrete labels), an
  example or two of how the client describes it on calls or in materials.

# Testimonials

Quotes and video testimonials from past clients. The writing agent uses these
as proof points and as language samples (how clients actually describe results).

## Conventions

- One file with all testimonials, organized by client name. Filename:
  `_all-testimonials.md` (or split per-client if the volume gets large).
- Each entry: client name (or anonymized), the quote (verbatim), date if known,
  context (what they hired the client for).

# 20-client-assets — User-Managed

This folder is yours. The data-gathering agent never reads from or writes to
anything in here. Drop client-supplied materials in the relevant subfolder and
keep this index up to date so the writing agent knows what's where.

## Subfolders

- [`frameworks/`](frameworks/) — named methodologies and frameworks the client owns
- [`case-studies/`](case-studies/) — past client wins with specifics (numbers, outcomes)
- [`testimonials/`](testimonials/) — quotes / video testimonials from past clients
- [`sales-pages/`](sales-pages/) — markdown copies of the client's current landing/sales pages

## Conventions

- One concept per file — keep individual files focused so the writing agent can
  pull just what it needs.
- File naming: lowercase, dashes, descriptive. e.g. `4-pillar-tax-method.md`,
  `2025-09-jane-doe-saved-187k.md`.
- Update this `_index.md` (and each subfolder's `_index.md`) when you add files,
  with a one-line description of each.
